# proyecto remate

## Catálogo en línea: Los usuarios pueden explorar productos, filtrarlos por categorías, revisar disponibilidad y añadirlos al carrito.

## Dashboard administrativo: Permite visualizar estadísticas de productos y del carrito para un control rápido del inventario.

## Gestión de productos (CRUD): Posibilidad de crear, editar y eliminar productos, definiendo nombre, descripción, precio, categoría y stock.

## Gestión de categorías: Permite agregar y eliminar categorías, manteniendo integridad con los productos existentes.

## Autenticación de administrador: Acceso seguro mediante login (usuario: admin / contraseña: admin123) para habilitar funciones de administración.
